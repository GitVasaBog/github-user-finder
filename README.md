# GitHub User Finder

**Автор:** [Ваше имя и фамилия]

## Описание проекта
Приложение для поиска пользователей GitHub с графическим интерфейсом. Позволяет искать пользователей, просматривать их информацию и сохранять в избранное.

## Функционал
- Поиск пользователей GitHub по логину
- Отображение информации (имя, компания, локация, репозитории, подписчики)
- Добавление пользователей в избранное
- Сохранение избранного в JSON файл
- Проверка пустого поля поиска

## Как использовать GitHub API
Приложение использует GitHub REST API v3:
- **Endpoint:** `https://api.github.com/users/{username}`
- **Метод:** GET
- **Ответ:** JSON с информацией о пользователе

## Пример использования
1. Запустите программу: `python main.py`
2. Введите имя пользователя (например, `octocat`)
3. Нажмите "Найти"
4. Нажмите "Добавить в избранное" для сохранения
5. Закройте и откройте программу снова - избранное сохранится
